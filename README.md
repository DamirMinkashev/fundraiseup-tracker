# fundraiseup-tracker

Тестовое задание для Fundraiseup.

# Запуск
1. ```npm i```
2. ```npm run client```
3. ```npm run server```

# Подключение к бд

По дефолту коннект будет происходить по адресу ```mongodb://127.0.0.1:27017/tracker```. Чтобы указать другой адрес, создайте ```.env``` файл с полем ```MONGODB_URL```
